# Karuuta Corpus

狩歌ビンゴ、狩歌大富豪の最適な選曲を研究するリポジトリ

## init

nodenvの利用を前提にしています。これを利用したくない場合は、`.node-version`ファイルに書かれたバージョンのNode.js実行環境を導入してください。
(この場合、`nodenv install`は不要)

```
$ git clone git@github.com:Arthur1/karuuta-corpus.git
$ cd karuuta-corpus
$ nodenv install
$ yarn install
```

## run

```
$ yarn main
```
